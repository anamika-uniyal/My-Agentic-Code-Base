# 🚀 GitHub Copilot Salesforce

This repository demonstrates how GitHub Copilot can assist Salesforce developers in:

- Writing Apex classes
- Generating test classes
- Improving code quality
- Performing code reviews
- Explaining logic

## Demo Flow

1. Generate Apex Service Class
2. Create Trigger & Handler
3. Generate Test Class
4. Perform Code Review
5. Optimize Code

## Tech Stack

- Salesforce Apex
- SOQL
- GitHub Copilot
